# Game Tracker Public Frontend Roadmap

Scope confirmed on 2026-05-11: implement items 1-9 for the public frontend. Item 10 is explicitly out of scope for now.

## Accepted Items

1. Local watchlist: let public visitors save games in browser storage and revisit them from a dedicated page.
2. Stronger dashboard filters: rating multi-select, release window, price, tags, and practical sorting modes.
3. Search completion: richer command-style search with covers, ratings, metadata, keyboard navigation, and empty-state recommendations.
4. Game detail decision summary: make the detail page explain why a game is worth watching before deep charts.
5. Released-game opinion view: help readers scan launch reception, review volume, language mix, and topic sentiment.
6. Release calendar improvements: improve monthly scanning, date selection, and watchlist-aware discovery.
7. Public API readiness: keep frontend data access switchable through public API base configuration and document assumptions.
8. SEO and sharing: metadata, dynamic titles, Open Graph-ready descriptions, and index-friendly public pages.
9. Loading and cache polish: keep the public experience fast, resilient, and clear while data is being fetched.

## Deferred

10. Opening content rankings or overseas trend modules to the public version is deferred until product scope is confirmed.
